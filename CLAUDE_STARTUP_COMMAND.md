# Claude 2.6 AI Initialization Command

To initialize and start the SpotifyRandom project using Claude 2.6 AI, please follow the steps below:

## Prerequisites
- Ensure you have Python 3.8 or higher installed.
- Install the necessary dependencies by running the following command:
  ```bash
  pip install -r requirements.txt
  ```

## Running the Initialization Command

To start the project, use the following command in your terminal:
```bash
claude start --project SpotifyRandom
```

This will initialize the project and start Claude with the required configurations to work with SpotifyRandom.
